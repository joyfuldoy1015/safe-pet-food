# 제품 우선 탐색 시나리오 분석

## 🎯 실제 사용자 시나리오

### **시나리오 1: 구체적인 제품 검색**
```
사용자: "로얄캐닌 어덜트 후기가 궁금해"

현재 경로:
1. 홈 접속
2. ??? (어디로 가야 할까?)
3. /brands → 로얄캐닌 찾기 → 제품 목록 → 어덜트 찾기
   (3단계, 복잡함)

원하는 경로:
1. 검색창에 "로얄캐닌 어덜트" 입력
2. 바로 제품 상세 페이지
   (1단계, 간단함)
```

### **시나리오 2: 카테고리 탐색**
```
사용자: "성견용 사료 중에 뭐가 좋을까?"

현재 경로:
1. /brands → 브랜드 선택 → 제품 목록 → 성견용 찾기
2. 다시 /brands → 다른 브랜드 → 제품 목록 → 성견용 찾기
   (반복, 비효율적)

원하는 경로:
1. 제품 목록 → 필터: "성견용"
2. 모든 브랜드의 성견용 사료 비교
   (1단계, 효율적)
```

### **시나리오 3: 등급별 탐색**
```
사용자: "A등급 제품만 보고 싶어"

현재 경로:
브랜드별로 하나씩 들어가서 확인
(매우 비효율적)

원하는 경로:
제품 목록 → 필터: "A등급"
(즉시 확인)
```

---

## 🔍 현재 구조의 문제점

### **❌ 제품 우선 사용자의 불편함**

```
1. 홈 화면 CTA:
   - "브랜드 투명성" ← 브랜드 중심
   - "펫 로그" ← 커뮤니티
   - "Q&A 포럼" ← 커뮤니티
   
   ❌ 제품 직접 검색 경로 없음!

2. 헤더 메뉴:
   - "브랜드/제품" ← /brands 연결 (브랜드 목록만)
   - "펫 로그"
   - "커뮤니티"
   
   ❌ 제품 목록 직접 접근 불가!

3. /search 존재하지만:
   ❌ 홈에서 눈에 띄지 않음
   ❌ 직접 URL 입력해야 함
```

## ✅ 해결 방안

### **Option 1: `/search`를 제품 검색 허브로 강화** ⭐⭐⭐ 최고 추천

#### **1-1. 홈 화면 CTA 추가**
```typescript
// components/home/FeatureCards.tsx

{
  title: '제품 검색',
  description: '다양한 브랜드의 제품을 비교하고 후기를 확인하세요',
  icon: Search,
  href: '/search?tab=products', // 제품 탭 기본
  color: 'from-purple-500 to-pink-500'
}
```

#### **1-2. 헤더 메뉴 개선**
```typescript
// app/components/Header.tsx

// 기존
'브랜드/제품' → /brands (브랜드만)

// 개선
'브랜드/제품' → 드롭다운
  - 브랜드 둘러보기 → /brands
  - 제품 검색하기 → /search?tab=products
  - 제품 전체보기 → /products (새로 만들기)
```

#### **1-3. `/search` 제품 탭 개선**
```typescript
// app/search/page.tsx

제품 탭에 추가:
- 인기 제품 TOP 10
- 최근 리뷰 많은 제품
- 등급별 필터 (A/B/C/D/F)
- 카테고리 필터 (사료/간식/영양제)
- 반려동물 필터 (강아지/고양이)
- 가격대 필터
- 정렬 (평점/추천/리뷰/최신)
```

**장점:**
- ✅ 기존 `/search` 활용 (추가 개발 최소)
- ✅ 브랜드/제품 통합 검색 가능
- ✅ 탭 전환으로 유연한 탐색


**장점:**
- ✅ 사용자 선택권
- ✅ URL 직관성
- ✅ 개발 부담 적음 (리다이렉트 또는 컴포넌트 재사용)

## 🎯 추천 방안

### **단계별 구현** ⭐⭐⭐

#### **Phase 1: 빠른 개선 (1일)**
```typescript
1. 홈 CTA 추가
   "제품 검색하기" → /search?tab=products

2. 헤더 메뉴 텍스트 변경
   "브랜드/제품" → "브랜드 둘러보기"
   새로 추가: "제품 검색" → /search?tab=products

3. /search 제품 탭 기본 필터 추가
   - 등급 필터
   - 정렬 (평점순/추천순)
```

#### **Phase 2: 본격 개선 (3일)**
```typescript
1. /products 페이지 생성
   - 필터/정렬 UI
   - 그리드 레이아웃
   - 페이지네이션

2. 고급 필터
   - 카테고리
   - 반려동물 종
   - 가격대
   - 인증

3. 제품 비교 기능
   - 최대 3개 제품 선택
   - 나란히 비교
```
