# 🎯 제품 리뷰/평가 작성 가이드

## 📋 작성 페이지 접근 방법

### 로그인 후:
1. **프로필 페이지**: `http://localhost:3000/profile` → "급여 후기 작성하기"
2. **펫 로그 페이지**: `http://localhost:3000/pet-log` → 우측 하단 "+" 버튼

---

## ✍️ 제품 리뷰/평가 작성 항목

### 1️⃣ 기본 정보
```
- 반려동물 선택: (드롭다운)
- 카테고리: 사료/간식/영양제/화장실
- 브랜드: 로얄캐닌
- 제품명: 로얄캐닌 어덜트
- 상태: 급여중/일시중단/급여종료
- 급여 시작일: 2024-01-01
- 급여 종료일: (선택)
```

## 💾 저장 후 자동 반영

### ✅ 제품 상세 페이지에 실시간 반영됨!

작성한 리뷰가 자동으로:
1. **소비자 평가** 섹션에 평균 점수로 집계
2. **추천/비추천** 섹션에 카운트 추가
3. **리뷰 목록**에 표시 (반려동물 정보 포함)

## 🔑 테스트 계정

### Magic Link (권장):
```
1. http://localhost:3000
2. "로그인" 클릭
3. 본인 이메일 입력
4. Magic Link 받기 → 이메일 확인 → 클릭
```

### 직접 생성 (Supabase):
```
1. Supabase 대시보드 → Authentication → Users
2. "Create new user" 클릭
3. 이메일/비밀번호 입력
4. 해당 계정으로 로그인
```

## 💡 참고사항

- **로그인 필수**: 제품 평가는 로그인 후에만 가능
- **반려동물 필수**: 최소 1마리 등록 필요
- **자동 저장**: 작성 즉시 DB 저장
- **실시간 반영**: 제품 상세 페이지에 바로 표시
- **수정 가능**: 작성 후 수정/삭제 가능 (본인만)
