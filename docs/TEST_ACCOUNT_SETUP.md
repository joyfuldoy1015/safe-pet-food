## 📋 테스트 계정 정보

```
이메일: user@test.com
비밀번호: user123
```

---

## 🚀 설정 방법

### 1단계: Supabase SQL 실행

1. **Supabase 대시보드** 접속
   - https://supabase.com/dashboard

2. **프로젝트 선택**
   - Safe Pet Food 프로젝트

3. **SQL Editor 열기**
   - 좌측 메뉴 → SQL Editor

4. **SQL 스크립트 실행**
   ```
   scripts/create-test-account.sql 파일의 내용 복사
   → SQL Editor에 붙여넣기
   → RUN 버튼 클릭
   ```

---

## ✅ 설정 완료 후 테스트

### 로그인 테스트

```
1. http://localhost:3000 접속
2. 우측 상단 "로그인" 버튼 클릭
3. 이메일: user@test.com 입력
4. 비밀번호: user123 입력
5. "로그인" 버튼 클릭
```

### 로그인 성공 확인

```
✅ 우측 상단에 "테스트유저" 닉네임 표시
✅ 프로필 드롭다운 메뉴 표시
✅ "프로필", "로그아웃" 메뉴 보임
```

## 🐛 문제 해결

### "Invalid login credentials" 오류
```
→ SQL 스크립트가 제대로 실행되지 않음
→ Supabase SQL Editor에서 다시 실행
```

### "Email not confirmed" 오류
```
→ SQL의 email_confirmed_at 값 확인
→ NOW()로 설정되어 있는지 확인
```

### 로그인 후 프로필이 안 보임
```
→ public.profiles 테이블 확인
→ user_id가 일치하는지 확인
```

## 💡 추가 테스트 계정 생성

다른 계정이 필요하면 SQL의 값을 변경:

```sql
-- 이메일 변경
'user2@test.com'

-- UUID 변경 (새로운 UUID 생성)
'a1b2c3d4-e5f6-4789-a012-3456789abcde'

-- 비밀번호 변경
crypt('newpassword123', gen_salt('bf'))

-- 닉네임 변경
'테스트유저2'
```
